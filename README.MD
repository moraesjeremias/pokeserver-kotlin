# Kotlin PokeServer
A Kotlin backend service for PokeAPI

## Technologies
- Kotlin
- Ktor
- Koin
- Kotlin Logging
- Config4k

- [x] Setup Routes (WIP)
- [ ] Setup Transactional Database

- [ ] Setup Github Actions CI/CD pipeline

## Setup and Run
Add pokeApi host to Environment Variables:

```POKEAPI_HOST=pokeapi.co/api/v2```
